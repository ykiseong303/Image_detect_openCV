### 프로젝트 명 : 장애물 표지판인식을 위한 딥러닝, 영상처리 프로젝트
+ 기간 : 2018.10 ~ 2018.12
+ 설명 : 
로봇 경진대회 프로젝트의 연장선으로, 표지판 인식을 위한 딥러닝과 다양한 영상처리 기법을 다룬 프로젝트입니다.  
도형인식과 색영역 추출, SIFT알고리즘, 합성곱 신경망 등을 이용한 영상처리 프로젝트입니다. 

+ 사용 데이터 : 
  + robotis 제공 표지판 이미지
  
+ 주사용기술 : 
  + python3
  + openCV
  + tensorflow
  + CNN (inception v3)
